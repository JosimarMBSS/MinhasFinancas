# Prompt base para GitHub Copilot / GitHub Models

Crie um app Android nativo chamado "Minhas Finanças" usando Java + XML, SQLite para persistência local e Material Design 3 para interface baseada em Views.

Objetivos:
- controle financeiro pessoal offline
- UI limpa e elegante
- categorias customizadas
- recorrências avançadas
- relatórios por período, categoria, tipo e status

Prioridades:
1. manter arquitetura por camadas
2. preservar base local com SQLite
3. separar data, repository, ui e util
4. criar fluxo sólido para CRUD de lançamentos e categorias
5. preparar entidades para recorrências e relatórios
6. evitar WebView e Capacitor
